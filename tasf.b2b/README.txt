En la presente carpeta, se desarrollará el proyecto de diseño y desarrollo
de software que involucra la implementación de un sistema para asignar maletas a vuelos de aerolíneas considerando ventanas de tiempo y capacidad de almacenaje.

Se involucra el desarrollo de 2 algoritmos metaheurísticos para concretar dicho fin y compararlos mediante experimentación numérica.

También, se realizan interfaces priorizando la visualización del mapa interactivo y las maletas con colores de semáforo según estados definidos del vuelo.
